  • @swervo - A short while after

    All you need to do is download bootforce.js and include it in your web page. Then add the appropriate data-target="#id" tags to your markup, give the corresponding elements the correct ids and it will 'just work'. If you're unsure it's just like Bootstrap the popular front-end development framework, refer to their documentation to get the hang of how it works. This site is built using bootforce incidentally.
